排程管理

排程管理的功能可以達到定時執行某項功能，像是開機、重啟、關機或者是備份和在控制台執行對應指令。

建立排程

進入排程管理後，點選「創建排程」按鈕。

輸入想給排程命名的名稱

以下部分是指排程多久要執行一次，分別有分、時、日、月、星期可以設置，以 Cronjob 格式執行，完成後點擊「創建排程」。 可以透過 https://crontab.guru 來預覽設置您的排程多久會執行。

CronJob 範例

進入該排程後，點擊右上方的「新排程」

創建任務有三個欄位

  1. 動作
    1. 發送指令（即在控制台發送指令）
    2. 發送電源指令（開機、重啟、關機）
    3. 建立備份
  2. 延遲（時間偏移），表示與上一個排程內的任務需要延遲多久才執行。
  3. 指令

都完成後點擊創建任務。
